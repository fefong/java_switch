# Java Switch Case
Example Application: Switch Case

## Java SwitchCase Example
```java
switch (option) {
case 0:
	// TODO
	System.out.println("Case: 0");
	break;
case 1:
	// TODO
	System.out.println("Case: 1");
	break;
default:
	// TODO
	System.out.println(String.format("Case: Default (value: %d)", option));
	break;
}
```
